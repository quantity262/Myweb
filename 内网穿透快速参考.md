# 内网穿透快速参考

## 🚀 最快方案（5分钟）

### ngrok（推荐新手）

1. **下载**：https://ngrok.com/download
2. **注册**：https://ngrok.com（免费）
3. **配置**：
   ```powershell
   .\ngrok.exe config add-authtoken 你的token
   ```
4. **启动**：
   ```powershell
   .\ngrok.exe http 3000
   ```
5. **访问**：使用显示的 `https://xxxx.ngrok.io` 地址

---

## 🇨🇳 国内推荐（速度快）

### cpolar（推荐国内用户）

1. **注册**：https://www.cpolar.com
2. **下载**：客户端
3. **登录**：使用账号登录
4. **创建隧道**：
   - 协议：HTTP
   - 本地地址：127.0.0.1:3000
5. **启动**：点击"启动"
6. **访问**：使用显示的地址

## ⚡ 一键启动（Windows）

使用项目中的 `启动内网穿透-ngrok.bat` 文件：
